Application de gestion de tickets de support DSI


Objectif

L’objectif est de créer une base de données à l’aide du langage JavaScript et de la relier à un site pour mettre en place un outil de support pour le service informatique.


Contexte

Ce projet est réalisé de manière autonome pendant les cours de TP et TD dans le cadre de la matière SAE23, le projet a été effectué pendant le deuxième semestre de la première année de BUT Réseaux et Télécommunications.


Travail réalisé

Pour réaliser ce travail, j’ai dû comprendre le code fourni pour le reproduire et l'adapter en fonction des erreurs présentes lors de l'écriture de mon code. J'ai également appris à utiliser MongoDB Compass pour ma base de données et à la connecter au code.


Résultat

À la fin de ce projet, mon application de gestion de tickets est fonctionnelle. Vous pouvez télécharger le dossier .ZIP qui contient tous mes fichiers pour ce projet.
